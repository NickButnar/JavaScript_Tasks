Suppose you have an array of strings representing user names in a system. Write a JavaScript program that prompts the user for a name and checks if such a name is in the array. If the name is found, display the message "Welcome, {name}!", and if it is not found, display "User not found."

Допустим, у вас есть массив строк, представляющих имена пользователей в системе. Напишите программу на JavaScript, которая запрашивает у пользователя имя и проверяет, есть ли такое имя в массиве. Если имя найдено, то выводится сообщение "Добро пожаловать, {имя}!", если не найдено - "Пользователь не найден".
