Suppose you have an array of strings representing colors. Write a JavaScript program that finds the number of different colors in the array and displays it on the screen.

Example array:
let colors = ["red", "green", "blue", "red", "yellow", "green", "red", "blue", "purple"];

Expected output:
The number of different colors is: 5

----------------------

Допустим, у вас есть массив строк, представляющих цвета. Напишите программу на JavaScript, которая находит количество различных цветов в массиве и выводит это число на экран.

Пример массива:
let colors = ["red", "green", "blue", "red", "yellow", "green", "red", "blue", "purple"];

Ожидаемый результат:
Количество различных цветов: 5
