Write a function filterRange(arr, a, b) that takes an array arr, finds elements with values greater than or equal to a and less than or equal to b, and returns the result as an array.
The function should return a new array and not modify the original one.

----------------

Напишите функцию filterRange(arr, a, b), которая принимает массив arr, ищет элементы со значениями больше или равными a и меньше или равными b и возвращает результат в виде массива.
Функция должна возвращать новый массив и не изменять исходный.
