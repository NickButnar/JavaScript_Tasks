Suppose you have an array of integers. Write a JavaScript program that finds the arithmetic mean of the elements in the array and displays it on the screen.

Example:
let numbers = [3, 6, 2, 8, 1, 9, 4];

Expected output:
The arithmetic mean of the array elements is: 4.857142857142857

---------------------

Допустим, у вас есть массив целых чисел. Напишите программу на JavaScript, которая находит среднее арифметическое значение элементов массива и выводит его на экран.

Например:
let numbers = [3, 6, 2, 8, 1, 9, 4];

Ожидаемый результат:
Среднее арифметическое значение элементов массива: 4.857142857142857
