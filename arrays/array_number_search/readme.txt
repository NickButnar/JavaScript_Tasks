Given an array of integers, find all the numbers that are repeated in the array and print them in ascending order. If there are no such numbers, print "NO".

Дан массив целых чисел. Найдите все числа, которые повторяются в массиве, и выведите их в порядке возрастания. Если таких чисел нет, выведите сообщение "NO".
